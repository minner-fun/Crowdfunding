```bash
npx tailwindcss init -p
npm start
```

## 📱 使用指南

### 1. 连接钱包
- 安装 MetaMask 浏览器扩展
- 切换到 Sepolia 测试网
- 点击"连接钱包"按钮

### 2. 浏览项目
- 在首页查看所有项目
- 使用搜索和过滤功能
- 点击项目卡片查看详情

### 3. 创建项目
- 点击导航栏"创建项目"
- 填写项目信息
- 确认交易并等待确认

### 4. 投资项目
- 选择感兴趣的项目
- 点击"立即投资"
- 输入投资金额并确认

### 5. 管理项目
- 在"我的项目"页面查看自己的项目
- 查看项目统计和投资者信息
